# 8-bit ALU using Verilog

## Overview

Designed and simulated an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Features

- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Left Shift
- Right Shift

## Flags

- Carry
- Zero
- Overflow

## Tools Used

- Verilog HDL
- ModelSim

## Files

- alu.v
- alu_tb.v

## Simulation

Successfully verified using a Verilog testbench and waveform simulation.
